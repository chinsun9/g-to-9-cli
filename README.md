# g-to-9-cli

> replace 'g' to '9'

## Install

```sh terminal
npm install -g g-to-9-cli
```

## Usage

```sh terminal
$ gto9 --help
Usage: cli [options] <text>

replace all 'g' to '9'

Examples:
  $ gto9 chinsung
  $ gto9 gg is good game
  $ gto9 'gg is good game'

Options:
  -V, --version  output the version number
  -h, --help     display help for command
```
